# 2022 KAKAO TECH INTERNSHIP - 코딩 테스트 공부 해설

본 문서는 **2022 KAKAO TECH INTERNSHIP**에 출제된 **코딩 테스트 공부** 문제의 상세한 알고리즘 분석, 풀이 전략 및 3개 언어(Python, C++, Java)의 모범 답안과 실행 코드를 제공합니다.

---

## 1. 문제 분석

### 알고리즘 유형
- **동적 계획법 (Dynamic Programming, DP)**
- **다익스트라 알고리즘 (Dijkstra's Algorithm - 최단 경로 유형으로도 접근 가능하나 DP 테이블 채우기가 더 정석적이고 빠름)**

### 문제의 핵심 및 난이도
- 현재 보유한 `알고력(alp)`과 `코딩력(cop)`이 주어지고, 주어진 모든 문제를 풀 수 있는 조건을 달성하기 위한 **최소 소요 시간**을 구하는 문제입니다.
- 모든 문제를 풀 수 있는 상태란, 주어진 문제들 중 **요구하는 알고력의 최댓값(`max_alp`)**과 **요구하는 코딩력의 최댓값(`max_cop`)**을 만족시키는 상태를 뜻합니다.
- **상태의 전이(Transition)**:
  1. 알고리즘 공부를 한다: 알고력 +1, 시간 +1
  2. 코딩 공부를 한다: 코딩력 +1, 시간 +1
  3. 현재 풀 수 있는 문제를 푼다: 알고력 +`alp_rwd`, 코딩력 +`cop_rwd`, 시간 +`cost`
- 난이도는 프로그래머스 **Level 3**로, 2차원 DP 테이블을 채워나갈 때 범위를 벗어나는 인덱스 처리(오버플로우)를 어떻게 깔끔하게 보정해 주느냐가 효율성과 정확성 테스트 통과의 최대 관건입니다.

## 2. 단계별 풀이 전략

### [1단계] 목표 알고력과 코딩력 계산
- 주어진 모든 문제를 풀기 위해 도달해야 하는 최소한의 알고력과 코딩력의 목표치(`max_alp`, `max_cop`)를 구합니다.
- 만약 초기의 알고력(`alp`) 또는 코딩력(`cop`)이 이미 이 목표값보다 높은 경우가 있을 수 있으므로 예외 처리를 해줍니다.
  - `alp = min(alp, max_alp)`
  - `cop = min(cop, max_cop)`

### [2단계] DP 테이블 설계 및 초기화
- `dp[i][j]`를 **알고력 `i`, 코딩력 `j`에 도달하기까지 걸리는 최소 시간**으로 정의합니다.
- 테이블의 크기는 `(max_alp + 1) * (max_cop + 1)`이 됩니다.
- 모든 값을 아주 큰 값(무한대, 예: `1e9`)으로 초기화한 뒤, 초기 상태인 `dp[alp][cop] = 0`으로 설정합니다.

### [3단계] 2차원 DP 테이블 전이 연산
- $i$는 `alp`부터 `max_alp`까지, $j$는 `cop`부터 `max_cop`까지 순회합니다.
- **1. 단순히 1씩 늘리는 학습**:
  - 알고력 공부: `dp[i + 1][j] = min(dp[i + 1][j], dp[i][j] + 1)` (단, $i < \text{max\_alp}$인 경우만)
  - 코딩력 공부: `dp[i][j + 1] = min(dp[i][j + 1], dp[i][j] + 1)` (단, $j < \text{max\_cop}$인 경우만)
- **2. 현재 상태에서 풀 수 있는 문제 풀기**:
  - 각 문제 `[alp_req, cop_req, alp_rwd, cop_rwd, cost]`를 순회합니다.
  - 현재 상태 `(i, j)`가 문제를 풀 수 있는 조건(`i >= alp_req` 이고 `j >= cop_req`)을 만족하는지 확인합니다.
  - **오버플로우 방지 보정**: 문제를 풀어 증가한 능력치 `(i + alp_rwd, j + cop_rwd)`가 우리의 최종 목표치(`max_alp`, `max_cop`)를 초과할 경우, 목표치 이상은 전부 목표치에 도달한 동일한 가치로 판단해야 하므로 **목표치 크기로 값을 클리핑(Clipping)** 해줍니다.
    - `next_alp = min(max_alp, i + alp_rwd)`
    - `next_cop = min(max_cop, j + cop_rwd)`
  - 상태 갱신: `dp[next_alp][next_cop] = min(dp[next_alp][next_cop], dp[i][j] + cost)`

### [4단계] 최적의 해 도출
- 테이블 갱신이 완료되면 모든 목표를 충족시킨 상태인 `dp[max_alp][max_cop]`의 값을 반환합니다.

## 3. 코드 구현 (Python, C++, Java)

### 🐍 Python 코드
사용자 인터랙티브 콘솔 입력을 지원하고 효율성 기준에 부합하게 구현한 모범 코드입니다.

```python
def solution(alp, cop, problems):
    # 1. 도달해야 하는 목표 알고력(max_alp)과 코딩력(max_cop) 구하기
    max_alp = 0
    max_cop = 0
    for p in problems:
        max_alp = max(max_alp, p[0])
        max_cop = max(max_cop, p[1])
        
    # 초기의 알고력/코딩력이 이미 목표치를 초과한 경우 예외 처리 (클리핑)
    alp = min(alp, max_alp)
    cop = min(cop, max_cop)
    
    # 2. DP 테이블 초기화 (충분히 큰 값인 무한대로 설정)
    INF = int(1e9)
    dp = [[INF] * (max_cop + 1) for _ in range(max_alp + 1)]
    
    # 시작 상태는 시간이 0 소요됨
    dp[alp][cop] = 0
    
    # 3. 2차원 DP 테이블 채우기
    for i in range(alp, max_alp + 1):
        for j in range(cop, max_cop + 1):
            # 단순히 1씩 수동으로 훈련하는 전이
            if i < max_alp:
                dp[i + 1][j] = min(dp[i + 1][j], dp[i][j] + 1)
            if j < max_cop:
                dp[i][j + 1] = min(dp[i][j + 1], dp[i][j] + 1)
                
            # 문제들을 풀어 더 빠르게 능력을 키우는 전이
            for alp_req, cop_req, alp_rwd, cop_rwd, cost in problems:
                # 현재 능력(i, j)으로 해당 문제를 풀 수 있는지 조건 검사
                if i >= alp_req and j >= cop_req:
                    # 오버플로우 방지 처리: 목표치를 초과한 능력치는 목표치 값으로 보정해 줍니다.
                    next_alp = min(max_alp, i + alp_rwd)
                    next_cop = min(max_cop, j + cop_rwd)
                    # 최솟값 테이블 갱신
                    dp[next_alp][next_cop] = min(dp[next_alp][next_cop], dp[i][j] + cost)
                    
    return dp[max_alp][max_cop]

# --- 직접 실행하여 결과를 검증할 수 있는 Main부 ---
if __name__ == "__main__":
    print("=== [Python] 코딩 테스트 공부 실행기 ===")
    try:
        alp = int(input("초기 알고력을 입력하세요: "))
        cop = int(input("초기 코딩력을 입력하세요: "))
        
        # 문제 파싱 예시: 10 15 2 1 2 , 20 20 3 3 4
        print("문제 정보들을 입력하세요. 문제 형태: [필요알고력 필요코딩력 보상알고력 보상코딩력 소요시간]")
        print("여러 문제는 세미콜론(;)으로 구분하여 입력하세요 (예: 10 15 2 1 2 ; 20 20 3 3 4): ")
        problems_input = input()
        
        problems = []
        for item in problems_input.split(';'):
            if item.strip():
                problems.append(list(map(int, item.split())))
                
        result = solution(alp, cop, problems)
        print(f"👉 목표 도달 최소 소요 시간: {result}시간")
    except ValueError:
        print("형식에 맞는 올바른 숫자를 입력하세요.")
```

### 🔵 C++ 코드
성능을 최대로 확보하기 위해 다차원 정적 벡터와 `std::min`을 이용해 구현한 C++ 전체 소스입니다.

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <sstream>

using namespace std;

// solution 함수: 목표 달성을 위한 최단시간을 구합니다.
int solution(int alp, int cop, vector<vector<int>> problems) {
    int max_alp = 0;
    int max_cop = 0;
    
    // 1. 필요한 최대 알고력과 코딩력을 산출합니다.
    for (int i = 0; i < problems.size(); i++) {
        max_alp = max(max_alp, problems[i][0]);
        max_cop = max(max_cop, problems[i][1]);
    }
    
    // 이미 현재 능력이 목표 능력을 넘었다면 목표 능력에 맞게 조율합니다.
    alp = min(alp, max_alp);
    cop = min(cop, max_cop);
    
    // 2. DP 테이블 초기화 (무한대 역할의 상수로 세팅)
    const int INF = 1e9;
    vector<vector<int>> dp(max_alp + 1, vector<int>(max_cop + 1, INF));
    dp[alp][cop] = 0;
    
    // 3. 2차원 동적계획법 테이블 갱신
    for (int i = alp; i <= max_alp; i++) {
        for (int j = cop; j <= max_cop; j++) {
            // 단순히 자습을 하는 경우
            if (i < max_alp) {
                dp[i + 1][j] = min(dp[i + 1][j], dp[i][j] + 1);
            }
            if (j < max_cop) {
                dp[i][j + 1] = min(dp[i][j + 1], dp[i][j] + 1);
            }
            
            // 문제를 푸는 경우
            for (int k = 0; k < problems.size(); k++) {
                int alp_req = problems[k][0];
                int cop_req = problems[k][1];
                int alp_rwd = problems[k][2];
                int cop_rwd = problems[k][3];
                int cost    = problems[k][4];
                
                // 요구조건을 통과하는 경우에만 갱신 가능
                if (i >= alp_req && j >= cop_req) {
                    int next_alp = min(max_alp, i + alp_rwd);
                    int next_cop = min(max_cop, j + cop_rwd);
                    dp[next_alp][next_cop] = min(dp[next_alp][next_cop], dp[i][j] + cost);
                }
            }
        }
    }
    
    return dp[max_alp][max_cop];
}

// --- 직접 실행하여 결과를 검증할 수 있는 Main 함수 ---
int main() {
    cout << "=== [C++] 코딩 테스트 공부 실행기 ===" << endl;
    
    int alp, cop;
    cout << "초기 알고력과 코딩력을 입력하세요 (공백 구분): ";
    if (!(cin >> alp >> cop)) return 0;
    
    int n;
    cout << "입력할 문제들의 개수를 입력하세요: ";
    cin >> n;
    
    vector<vector<int>> problems(n, vector<int>(5));
    cout << n << "개의 문제 정보들을 형식에 맞춰 차례대로 입력하세요." << endl;
    cout << "(형식: [필요알고력 필요코딩력 보상알고력 보상코딩력 소요시간])" << endl;
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < 5; j++) {
            cin >> problems[i][j];
        }
    }
    
    int result = solution(alp, cop, problems);
    cout << "👉 목표 도달 최소 소요 시간: " << result << "시간" << endl;
    
    return 0;
}
```

### ☕ Java 코드
Java 가비지 컬렉터 부담을 줄이기 위해 원시 2차원 배열과 빠른 점화식 갱신을 조합한 코드 파일입니다.

```java
import java.util.Scanner;
import java.util.Arrays;

public class Main {
    
    // solution 메소드: 최소 공부 시간을 구합니다.
    public static int solution(int alp, int cop, int[][] problems) {
        int max_alp = 0;
        int max_cop = 0;
        
        // 1. 달성해야 할 최대 요구 조건 탐색
        for (int i = 0; i < problems.length; i++) {
            max_alp = Math.max(max_alp, problems[i][0]);
            max_cop = Math.max(max_cop, problems[i][1]);
        }
        
        // 초기 능력이 이미 모든 목표를 달성한 경우의 조율
        alp = Math.min(alp, max_alp);
        cop = Math.min(cop, max_cop);
        
        // 2. DP 테이블 구축 및 초기화
        int INF = 1_000_000_000;
        int[][] dp = new int[max_alp + 1][max_cop + 1];
        for (int i = 0; i <= max_alp; i++) {
            Arrays.fill(dp[i], INF);
        }
        
        dp[alp][cop] = 0; // 시작점 설정
        
        // 3. 점화식 테이블 갱신
        for (int i = alp; i <= max_alp; i++) {
            for (int j = cop; j <= max_cop; j++) {
                // 수동 훈련
                if (i < max_alp) {
                    dp[i + 1][j] = Math.min(dp[i + 1][j], dp[i][j] + 1);
                }
                if (j < max_cop) {
                    dp[i][j + 1] = Math.min(dp[i][j + 1], dp[i][j] + 1);
                }
                
                // 문제를 통한 훈련
                for (int[] p : problems) {
                    int alp_req = p[0];
                    int cop_req = p[1];
                    int alp_rwd = p[2];
                    int cop_rwd = p[3];
                    int cost    = p[4];
                    
                    if (i >= alp_req && j >= cop_req) {
                        int next_alp = Math.min(max_alp, i + alp_rwd);
                        int next_cop = Math.min(max_cop, j + cop_rwd);
                        dp[next_alp][next_cop] = Math.min(dp[next_alp][next_cop], dp[i][j] + cost);
                    }
                }
            }
        }
        
        return dp[max_alp][max_cop];
    }

    // --- 직접 실행하여 결과를 검증할 수 있는 Main 메소드 ---
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("=== [Java] 코딩 테스트 공부 실행기 ===");
        
        System.out.print("초기 알고력을 입력하세요: ");
        int alp = scanner.nextInt();
        
        System.out.print("초기 코딩력을 입력하세요: ");
        int cop = scanner.nextInt();
        
        System.out.print("입력할 문제의 개수를 입력하세요: ");
        int n = scanner.nextInt();
        
        int[][] problems = new int[n][5];
        System.out.println(n + "개의 문제들을 형식에 맞게 입력하세요.");
        System.out.println("(형식: [필요알고력 필요코딩력 보상알고력 보상코딩력 소요시간])");
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < 5; j++) {
                problems[i][j] = scanner.nextInt();
            }
        }
        
        int result = solution(alp, cop, problems);
        System.out.println("👉 목표 도달 최소 소요 시간: " + result + "시간");
        
        scanner.close(); // 스캐너 종료
    }
}
```

## 4. 핵심 포인트 및 주의 사항

1. **인덱스 오버플로우 방지 (클리핑)**:
   - 문제를 풀어 증가한 능력치(`i + alp_rwd`, `j + cop_rwd`)가 목표치인 `max_alp`, `max_cop`보다 클 수 있습니다.
   - 이때 단순 2차원 배열의 인덱스로 접근하게 되면 `ArrayIndexOutOfBoundsException`이 발생하거나 테이블이 엉뚱한 크기로 커져야 합니다.
   - 따라서 목표치를 넘어서는 모든 상태는 **목표치 인덱스에 다 모아서 상태를 갱신**하도록 `min` 보정을 걸어주는 것(`min(max_alp, i + alp_rwd)`)이 이 문제의 핵심 열쇠입니다.

2. **초기값의 목표 도달 여부 예외 처리**:
   - `alp` 또는 `cop`가 처음부터 `max_alp` 또는 `max_cop`보다 크게 주어지는 엣지 케이스가 존재할 수 있습니다.
   - 이 경우 DP 테이블 크기를 잡거나 순회할 때 오류가 생기므로 루프를 시작하기 전에 `min(alp, max_alp)` 처리를 무조건 수행해야 합니다.

3. **시간 및 공간 복잡도**:
   - `max_alp`와 `max_cop`는 각각 최대 150입니다.
   - DP 테이블의 칸 수는 최대 $150 \times 150 = 22,500$칸입니다.
   - 한 칸당 최대 100개의 문제를 순회하므로 전체 연산 횟수는 대략 $22,500 \times 100 \approx 2.25 \times 10^6$번의 연산에 불과하여 시간 복잡도는 **$O(\text{max\_alp} \times \text{max\_cop} \times N)$**이며 약 5ms 이하의 초고속 효율성을 제공합니다.
