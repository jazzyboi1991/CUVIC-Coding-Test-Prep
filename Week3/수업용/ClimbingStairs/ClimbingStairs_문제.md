# LeetCode 70. Climbing Stairs (계단 오르기)

## 문제 설명

당신은 계단을 오르고 있습니다. 이 계단의 정상에 도달하려면 총 `n`개의 계단(steps)을 올라가야 합니다.

매번 계단을 오를 때마다 당신은 **1계단** 또는 **2계단**씩 올라갈 수 있습니다. 정상에 도달할 수 있는 **서로 다른 방법의 수**가 총 몇 가지인지 구하는 코드를 작성하세요.

---

## 제한사항

- $1 \le n \le 45$

---

## 입출력 예

| Input (n) | Output |
| :--- | :--- |
| `2` | `2` |
| `3` | `3` |

---

## 입출력 예 설명

### 입출력 예 #1
정상(`n = 2`)에 도달하는 방법은 다음과 같이 두 가지가 있습니다.
1. 1계단 + 1계단
2. 2계단

### 입출력 예 #2
정상(`n = 3`)에 도달하는 방법은 다음과 같이 세 가지가 있습니다.
1. 1계단 + 1계단 + 1계단
2. 1계단 + 2계단
3. 2계단 + 1계단
