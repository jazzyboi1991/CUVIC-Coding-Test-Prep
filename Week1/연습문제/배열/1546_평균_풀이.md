# 백준 1546 - 평균 풀이

## 풀이 전략
1. 과목의 개수 `N`을 입력받습니다.
2. 모든 점수를 입력받으며 최댓값 `M`과 점수들의 총합 `sum`을 구합니다.
3. 새로운 평균을 계산합니다.
    - 새로운 점수들의 합은 `(score1/M*100) + (score2/M*100) + ... + (scoreN/M*100)` 입니다.
    - 이는 `(score1 + score2 + ... + scoreN) / M * 100`과 같습니다.
    - 최종 평균은 `sum / M * 100 / N`이 됩니다.

## 코드 (C++)
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int N;
    cin >> N;
    
    vector<int> scores(N);
    int max_score = 0;
    double sum = 0;
    
    for (int i = 0; i < N; i++) {
        cin >> scores[i];
        if (scores[i] > max_score) max_score = scores[i];
        sum += scores[i];
    }
    
    cout << (sum / max_score * 100) / N << endl;
    
    return 0;
}
```

## 코드 (Python)
```python
N = int(input())
scores = list(map(int, input().split()))
max_score = max(scores)
new_average = sum(scores) / max_score * 100 / N
print(new_average)
```

## 코드 (Java)
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        
        long max = 0;
        long sum = 0;
        
        for (int i = 0; i < N; i++) {
            int temp = sc.nextInt();
            if (temp > max) max = temp;
            sum += temp;
        }
        
        System.out.println(sum * 100.0 / max / N);
    }
}
```
