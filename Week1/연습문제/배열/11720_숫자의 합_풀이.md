# 백준 11720 - 숫자의 합 풀이

## 풀이 전략
1. 숫자의 개수 `N`을 입력받습니다.
2. 둘째 줄에 공백 없이 주어지는 숫자들을 문자열로 입력받습니다.
3. 문자열의 각 문자를 숫자로 변환하여 합계에 더합니다.
    - C++의 경우 `s[i] - '0'`을 이용하거나 Python의 경우 `int(char)`를 사용할 수 있습니다.

## 코드 (C++)
```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    int N;
    cin >> N;
    
    string s;
    cin >> s;
    
    long long sum = 0;
    for (int i = 0; i < N; i++) {
        sum += s[i] - '0';
    }
    
    cout << sum << endl;
    
    return 0;
}
```

## 코드 (Python)
```python
N = int(input())
numbers = input()
print(sum(map(int, list(numbers))))
```

## 코드 (Java)
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        String sNum = sc.next();
        char[] cNum = sNum.toCharArray();
        long sum = 0;
        for (int i = 0; i < cNum.length; i++) {
            sum += cNum[i] - '0';
        }
        System.out.println(sum);
    }
}
```
