# 백준 1940 - 주몽 풀이

## 풀이 전략
1. 재료의 개수 $N$이 최대 15,000입니다. 2개를 뽑는 모든 경우의 수를 계산하면 $O(N^2)$으로 약 2억 2천만 번의 연산이 필요해 시간 초과 위험이 있습니다.
2. 따라서 데이터를 정렬한 후 **투 포인터(Two Pointer)** 알고리즘을 사용합니다.
3. 배열을 오름차순으로 정렬합니다 ($O(N \log N)$).
4. 양 끝을 가리키는 포인터 `i`와 `j`를 선언합니다.
   - `A[i] + A[j] < M`: 합이 작으므로 `i`를 오른쪽으로 한 칸 이동합니다.
   - `A[i] + A[j] > M`: 합이 크므로 `j`를 왼쪽으로 한 칸 이동합니다.
   - `A[i] + A[j] == M`: 갑옷을 만들 수 있습니다. `count`를 증가시키고 `i++`, `j--`를 수행합니다.
5. 전체 시간 복잡도는 $O(N \log N)$으로 해결 가능합니다.

## 코드 (C++)
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int n, m;
    cin >> n >> m;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    
    // 오름차순 정렬
    sort(a.begin(), a.end());
    
    int count = 0;
    int i = 0;
    int j = n - 1;
    
    // 투 포인터 이동
    while (i < j) {
        if (a[i] + a[j] < m) {
            i++;
        } else if (a[i] + a[j] > m) {
            j--;
        } else {
            count++;
            i++;
            j--;
        }
    }
    
    cout << count << endl;
    return 0;
}
```

import sys

n = int(sys.stdin.readline())

while i < j:
    s = arr[i] + arr[j]
    if s < m:
        i += 1
    elif s > m:
        j -= 1
    else:
        count += 1
        i += 1
        j -= 1

print(count)

## 코드 (Java)
```java
import java.util.*;
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(br.readLine());
        int m = Integer.parseInt(br.readLine());
        int[] arr = new int[n];
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        for (int i = 0; i < n; i++) {
            arr[i] = Integer.parseInt(st.nextToken());
        }
        
        // 정렬
        Arrays.sort(arr);
        
        int count = 0;
        int i = 0;
        int j = n - 1;
        
        // 투 포인터
        while (i < j) {
            int sum = arr[i] + arr[j];
            if (sum < m) {
                i++;
            } else if (sum > m) {
                j--;
            } else {
                count++;
                i++;
                j--;
            }
        }
        System.out.println(count);
    }
}
```
