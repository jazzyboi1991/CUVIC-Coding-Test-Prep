# 백준 12891 - DNA 비밀번호

- 시간 제한: 2초
- 메모리 제한: 512MB


## 문제
평소에 문자열을 가지고 노는 것을 좋아하는 민호는 DNA 문자열을 만들었다. DNA 문자열은 모든 문자가 'A', 'C', 'G', 'T'로 이루어진 문자열을 말한다.

민호는 자신이 만든 DNA 문자열의 부분문자열을 비밀번호로 사용하기로 하였다. 하지만 민호는 보안을 위해 임의의 DNA 문자열의 부분문자열을 뽑았을 때, 등장하는 문자의 개수가 특정 개수 이상이어야 비밀번호로 사용할 수 있다는 규칙을 만들었다.

예를 들어 임의의 DNA 문자열이 AAACCTGG이고, 뽑을 부분문자열의 길이를 4라고 하자. 그리고 A는 1개 이상, C는 1개 이상, G는 1개 이상, T는 0개 이상 등장해야 비밀번호로 사용할 수 있다고 하자.

이때, 처음에 뽑은 부분문자열 AAAC는 G가 0개라서 비밀번호로 쓸 수 없지만, 뒤로 한 칸 이동하여 만든 AACD는 G가 1개 이상이므로 비밀번호로 사용할 수 있다.

DNA 문자열의 길이 S와 비밀번호로 사용할 부분문자열의 길이 P가 주어졌을 때, 조건을 만족하는 비밀번호의 가지수를 구하는 프로그램을 작성하시오.

## 입력
첫째 줄에 DNA 문자열의 길이 S와 부분문자열의 길이 P가 주어진다. (1 ≤ P ≤ S ≤ 1,000,000)
둘째 줄에 DNA 문자열이 주어진다.
셋째 줄에 'A', 'C', 'G', 'T' 의 최소 개수가 공백을 사이에 두고 주어진다. 각 개수는 S보다 작거나 같은 음이 아닌 정수이며, 총 합은 S보다 작거나 같음이 보장된다.

## 출력
첫째 줄에 민호가 만들 수 있는 비밀번호의 종류의 수를 출력한다.

## 예제 입력 1
```
9 8
CCTGGATTG
2 0 1 1
```

## 예제 출력 1
```
0
```

## 예제 입력 2
```
4 2
GATA
1 0 0 1
```

## 예제 출력 2
```
2
```
