# 백준 2018 - 수들의 합 5 풀이

## 풀이 전략
1. 투 포인터(`start_index`, `end_index`)를 사용하여 연속된 자연수의 범위를 조절합니다.
2. 초기 상태: `sum = 1`, `count = 1`, `start_index = 1`, `end_index = 1`. (N 자신인 경우를 미리 count에 포함)
3. `sum`이 `N`과 같으면 `count`를 증가시키고 `end_index`를 오른쪽으로 이동합니다.
4. `sum`이 `N`보다 작으면 `end_index`를 오른쪽으로 이동하여 범위를 넓힙니다.
5. `sum`이 `N`보다 크면 `start_index`를 오른쪽으로 이동하여 범위를 좁힙니다.
6. `end_index`가 `N`에 도달할 때까지 반복합니다.

## 코드 (C++)
```cpp
#include <iostream>

using namespace std;

int main() {
    int N;
    cin >> N;

    int count = 1;
    int start_index = 1;
    int end_index = 1;
    int sum = 1;

    while (end_index != N) {
        if (sum == N) {
            count++;
            end_index++;
            sum = sum + end_index;
        } else if (sum > N) {
            sum = sum - start_index;
            start_index++;
        } else {
            end_index++;
            sum = sum + end_index;
        }
    }

    cout << count << endl;

    return 0;
}
```

## 코드 (Python)

```python
n = int(input())
count = 1

end_index = 1
sum_val = 1

while end_index != n:
    if sum_val == n:
        count += 1
        end_index += 1
        sum_val += end_index
    elif sum_val > n:
        sum_val -= start_index
        start_index += 1

        end_index += 1
```

## 코드 (Java)

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int count = 1;
        int start_index = 1;
        int end_index = 1;
        int sum = 1;
        
        while (end_index != N) {
            if (sum == N) {
                count++;
                end_index++;
                sum = sum + end_index;
            } else if (sum > N) {
                sum = sum - start_index;
                start_index++;
            } else {
                end_index++;
                sum = sum + end_index;
            }
        }
        System.out.println(count);
    }
}
```
