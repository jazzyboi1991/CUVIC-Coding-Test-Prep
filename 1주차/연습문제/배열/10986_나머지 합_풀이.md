# 백준 10986 - 나머지 합 풀이

## 풀이 전략
1. 구간 합 배열 `S`를 생성합니다.
2. 구간 합을 `M`으로 나눈 나머지들로 구성된 배열을 생각합니다.
    - 구간 `(i, j)`의 합 가 `M`으로 나누어 떨어진다는 것은 `(S[j] - S[i-1]) % M == 0` 임을 의미합니다.
    - 즉, `S[j] % M == S[i-1] % M` 인 쌍을 찾는 문제입니다.
3. 나머지가 같은 인덱스들 중에서 2개를 뽑는 경우의 수를 구합니다.
    - `C[R]`을 나머지가 `R`인 구간 합의 개수라고 할 때, `C[R] * (C[R] - 1) / 2`를 모두 더합니다.
    - 추가로, 나머지가 0인 경우는 그 자체로 조건을 만족하므로 `C[0]`을 더해줍니다.

## 코드 (C++)
```cpp
#include <iostream>
#include <vector>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int N, M;
    cin >> N >> M;

    vector<long> S(N + 1, 0);
    vector<long> C(M, 0);
    long long answer = 0;

    for (int i = 1; i <= N; i++) {
        int temp;
        cin >> temp;
        S[i] = S[i - 1] + temp;
        int remainder = S[i] % M;
        if (remainder == 0) answer++;
        C[remainder]++;
    }

    for (int i = 0; i < M; i++) {
        if (C[i] > 1) {
            answer = answer + (C[i] * (C[i] - 1) / 2);
        }
    }

    cout << answer << endl;

    return 0;
}
```

## 코드 (Python)
```python
import sys
input = sys.stdin.readline

n, m = map(int, input().split())
a = list(map(int, input().split()))

s = [0] * n
c = [0] * m
s[0] = a[0]
answer = 0

for i in range(1, n):
    s[i] = s[i-1] + a[i]

for i in range(n):
    remainder = s[i] % m
    if remainder == 0:
        answer += 1
    c[remainder] += 1

for i in range(m):
    if c[i] > 1:
        answer += (c[i] * (c[i] - 1) // 2)

print(answer)
```

## 코드 (Java)
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int M = sc.nextInt();
        long[] S = new long[N + 1];
        long[] C = new long[M];
        long answer = 0;
        
        for (int i = 1; i <= N; i++) {
            S[i] = S[i - 1] + sc.nextInt();
            int remainder = (int) (S[i] % M);
            if (remainder == 0) answer++;
            C[remainder]++;
        }
        
        for (int i = 0; i < M; i++) {
            if (C[i] > 1) {
                answer += (C[i] * (C[i] - 1) / 2);
            }
        }
        
        System.out.println(answer);
    }
}
```
