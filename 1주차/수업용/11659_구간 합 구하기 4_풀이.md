# 백준 11659 - 구간 합 구하기 4 풀이

## 풀이 전략
1. 수의 개수가 100,000이고 합을 구해야 하는 횟수가 100,000이므로 매번 합을 계산하면 시간 초과가 발생합니다.
2. 따라서 **누적 합(Prefix Sum)** 알고리즘을 사용합니다.
3. 배열 `A`의 i번째부터 j번째까지의 합은 `S[j] - S[i-1]`로 계산할 수 있습니다. (여기서 `S[i]`는 1번째부터 i번째까지의 합)
4. 누적 합 배열을 생성할 때 `S[i] = S[i-1] + A[i]` 공식으로 O(N)에 구합니다.
5. 이후 질의 답변은 O(1)에 수행하여 전체 시간 복잡도 O(N + M)으로 해결합니다.

## 코드 (C++)
```cpp
#include <iostream>
using namespace std;

int main() {
    // 입출력 속도 향상
    ios::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);

    int n, m;
    cin >> n >> m;
    
    // 누적 합 배열 (1-based index)
    int s[100001] = {0};
    for (int i = 1; i <= n; i++) {
        int temp;
        cin >> temp;
        s[i] = s[i - 1] + temp; // 이전 합 + 현재 수
    }

    for (int k = 0; k < m; k++) {
        int i, j;
        cin >> i >> j;
        // 구간 합 공식: S[j] - S[i-1]
        cout << s[j] - s[i - 1] << "\n";
    }
    
    return 0;
}
```

## 코드 (Python)
```python
import sys
input = sys.stdin.readline

n, m = map(int, input().split())
arr = list(map(int, input().split()))

# 누적 합 배열 만들기
prefix_sum = [0]
temp = 0
for i in arr:
    temp += i
    prefix_sum.append(temp)

# 질의에 따라 구간 합 계산
for _ in range(m):
    i, j = map(int, input().split())
    print(prefix_sum[j] - prefix_sum[i-1])
```

## 코드 (Java)
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        
        int n = Integer.parseInt(st.nextToken());
        int m = Integer.parseInt(st.nextToken());
        
        // 데이터가 크므로 8바이트 long형 배열 사용 권장 (현재 문제는 int로도 가능)
        long[] s = new long[n + 1];
        
        st = new StringTokenizer(br.readLine());
        for (int i = 1; i <= n; i++) {
            s[i] = s[i - 1] + Integer.parseInt(st.nextToken());
        }
        
        for (int q = 0; q < m; q++) {
            st = new StringTokenizer(br.readLine());
            int i = Integer.parseInt(st.nextToken());
            int j = Integer.parseInt(st.nextToken());
            
            // 구간 합 출력
            System.out.println(s[j] - s[i - 1]);
        }
    }
}
```
