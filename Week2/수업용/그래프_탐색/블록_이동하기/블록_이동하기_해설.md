# [해설] 블록 이동하기

## 문제 분석

이 문제는 `2 x 1` 크기의 로봇이 미로를 통과하여 `(N, N)` 좌표에 도달하는 **최단 시간**을 구하는 문제입니다. 모든 이동 비용이 1초로 동일하기 때문에 전형적인 **BFS (너비 우선 탐색)** 문제입니다.

**핵심 난이도 및 고려 사항**:

1. **상태 정의 (State Definition)**:
    - 일반적인 미로 찾기 문제에서는 (r, c) 좌표 하나로 현재 상태를 나타내지만, 이 문제의 로봇은 두 칸을 차지합니다.
    - 따라서 두 칸의 좌표를 묶어 하나의 상태로 정의해야 합니다. (예: `{(r1, c1), (r2, c2)}`)
2. **복잡한 이동 규칙**:
    - 상하좌우 평행 이동은 간단하지만, 회전(Rotation)이 추가되어 있습니다.
    - 특정 칸을 축으로 회전할 때, **회전 궤적 상에 벽이 없어야 한다**는 조건이 까다롭습니다.
3. **최단 거리 보장**:
    - 중복 방문을 피하기 위해 `visited` 처리를 해야 하는데, 로봇의 위치가 `{(1,1), (1,2)}`인 것과 `{(1,2), (1,1)}`인 것은 동일한 상태로 취급해야 합니다.

---

## 단계별 풀이 전략 (상세)

### 1. 지도 전처리 (Padding)

- 배열의 인덱스 범위를 매번 체크하는 번거로움을 피하기 위해, 입력받은 `board`의 상하좌우를 벽(`1`)으로 감싼 새로운 지도를 만듭니다. (크기: `(N+2) x (N+2)`)

### 2. 로봇의 위치 관리

- 두 칸의 좌표 세트를 `frozenset`이나 정렬된 튜플로 만들어 집합(`set`)에 넣음으로써 효율적으로 방문 체크를 수행합니다.

### 3. BFS 구현

- **큐 초기화**: `deque`를 사용하여 `(현재_위치, 경과_시간)`을 넣습니다. (시작: `{(1,1), (1,2)}`, `0초`)
- **이동 가능 위치 확인 (`get_next_pos`)**:
    - **평행 이동**: 상, 하, 좌, 우 4방향으로 로봇 전체를 이동시킵니다. 두 칸 모두 빈 공간이어야 합니다.
    - **회전 이동**:
        - 로봇이 **가로**일 때: 위쪽 두 칸이 비어있으면 위로 90도 회전(2가지), 아래쪽 두 칸이 비어있으면 아래로 90도 회전(2가지) 가능.
        - 로봇이 **세로**일 때: 왼쪽 두 칸이 비어있으면 왼쪽으로 90도 회전(2가지), 오른쪽 두 칸이 비어있으면 오른쪽으로 90도 회전(2가지) 가능.
- **방문 체크 및 큐 삽입**: 새로 이동할 위치가 `visited`에 없다면 큐에 넣고 방문 처리합니다.
- **종료 조건**: 로봇의 두 칸 중 하나라도 `(N, N)`에 도달하면 즉시 시간을 반환합니다.

## 코드 구현

### 1. Python

```python
from collections import deque

def get_next_pos(pos, board):
    next_pos = []
    pos = list(pos)
    r1, c1, r2, c2 = pos[0][0], pos[0][1], pos[1][0], pos[1][1]

    # 1. 상하좌우 이동
    dx = [-1, 1, 0, 0]
    dy = [0, 0, -1, 1]
    for i in range(4):
        nr1, nc1, nr2, nc2 = r1 + dx[i], c1 + dy[i], r2 + dx[i], c2 + dy[i]
        if board[nr1][nc1] == 0 and board[nr2][nc2] == 0:
            next_pos.append({(nr1, nc1), (nr2, nc2)})

    # 2. 회전
    if r1 == r2: # 가로일 때
        for i in [-1, 1]:
            if board[r1 + i][c1] == 0 and board[r2 + i][c2] == 0:
                next_pos.append({(r1, c1), (r1 + i, c1)})
                next_pos.append({(r2, c2), (r2 + i, c2)})
    elif c1 == c2: # 세로일 때
        for i in [-1, 1]:
            if board[r1][c1 + i] == 0 and board[r2][c2 + i] == 0:
                next_pos.append({(r1, c1), (r1, c1 + i)})
                next_pos.append({(r2, c2), (r2, c2 + i)})

    return next_pos

def solution(board):
    n = len(board)
    new_board = [[1] * (n + 2) for _ in range(n + 2)]
    for i in range(n):
        for j in range(n):
            new_board[i + 1][j + 1] = board[i][j]

    start_pos = {(1, 1), (1, 2)}
    q = deque([(start_pos, 0)])
    visited = [start_pos]

    while q:
        pos, dist = q.popleft()
        if (n, n) in pos:
            return dist

        for next_p in get_next_pos(pos, new_board):
            if next_p not in visited:
                q.append((next_p, dist + 1))
                visited.append(next_p)
    return 0

# 터미널 입력 예시
if __name__ == "__main__":
    n = int(input("지도의 크기(n)를 입력하세요: "))
    print("지도를 한 줄씩 입력하세요 (0: 빈칸, 1: 벽):")
    board = [list(map(int, input().split())) for _ in range(n)]
    print(f"결과: {solution(board)}")
```

### 2. C++

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <set>

using namespace std;

struct Robot {
    int r1, c1, r2, c2, dist;
};

vector<pair<pair<int, int>, pair<int, int>>> get_next_pos(int r1, int c1, int r2, int c2, vector<vector<int>>& board) {
    vector<pair<pair<int, int>, pair<int, int>>> next_pos;

    // 이동
    int dr[] = {-1, 1, 0, 0};
    int dc[] = {0, 0, -1, 1};
    for(int i=0; i<4; i++) {
        int nr1 = r1 + dr[i], nc1 = c1 + dc[i];
        int nr2 = r2 + dr[i], nc2 = c2 + dc[i];
        if(board[nr1][nc1] == 0 && board[nr2][nc2] == 0) {
            next_pos.push_back({{nr1, nc1}, {nr2, nc2}});
        }
    }

    // 회전
    if(r1 == r2) { // 가로
        for(int i : {-1, 1}) {
            if(board[r1+i][c1] == 0 && board[r2+i][c2] == 0) {
                next_pos.push_back({{r1, c1}, {r1+i, c1}});
                next_pos.push_back({{r2, c2}, {r2+i, c2}});
            }
        }
    } else { // 세로
        for(int i : {-1, 1}) {
            if(board[r1][c1+i] == 0 && board[r2][c2+i] == 0) {
                next_pos.push_back({{r1, c1}, {r1, c1+i}});
                next_pos.push_back({{r2, c2}, {r2, c2+i}});
            }
        }
    }
    return next_pos;
}

int solution(vector<vector<int>> board) {
    int n = board.size();
    vector<vector<int>> new_board(n + 2, vector<int>(n + 2, 1));
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            new_board[i + 1][j + 1] = board[i][j];

    queue<Robot> q;
    q.push({1, 1, 1, 2, 0});
    set<set<pair<int, int>>> visited;
    visited.insert(set<pair<int, int>>{{1, 1}, {1, 2}});

    while (!q.empty()) {
        Robot curr = q.front(); q.pop();
        if ((curr.r1 == n && curr.c1 == n) || (curr.r2 == n && curr.c2 == n)) return curr.dist;

        for (auto next : get_next_pos(curr.r1, curr.c1, curr.r2, curr.c2, new_board)) {
            set<pair<int, int>> pos = {next.first, next.second};
            if (visited.find(pos) == visited.end()) {
                q.push({next.first.first, next.first.second, next.second.first, next.second.second, curr.dist + 1});
                visited.insert(pos);
            }
        }
    }
    return 0;
}

int main() {
    int n;
    cout << "지도의 크기(n)를 입력하세요: ";
    cin >> n;
    vector<vector<int>> board(n, vector<int>(n));
    cout << "지도를 입력하세요 (0: 빈칸, 1: 벽):" << endl;
    for(int i=0; i<n; i++) {
        for(int j=0; j<n; j++) {
            cin >> board[i][j];
        }
    }
    cout << "결과: " << solution(board) << endl;
    return 0;
}
```

### 3. Java

```java
import java.util.ArrayList;   // 가변적인 리스트를 만들기 위한 클래스입니다.
import java.util.Arrays;      // 배열을 다루기 위한 유용한 도구들이 들어있는 라이브러리입니다.
import java.util.HashSet;     // 중복을 허용하지 않는 데이터 집합(Set)을 관리합니다.
import java.util.LinkedList;  // 큐(Queue) 기능을 구현하기 위해 사용되는 리스트입니다.
import java.util.List;        // 리스트 자료형의 표준 인터페이스입니다.
import java.util.Queue;       // BFS 탐색을 위한 '줄 서기(FIFO)' 형태의 자료구조입니다.
import java.util.Scanner;     // 사용자로부터 지도의 크기 등을 입력받기 위한 도구입니다.
import java.util.Set;         // 집합 자료형의 표준 인터페이스입니다.

// 로봇의 상태(두 칸의 좌표와 시간)를 저장하는 클래스입니다.
class Node {
    int r1, c1, r2, c2, dist;
    Node(int r1, int c1, int r2, int c2, int dist) {
        this.r1 = r1; this.c1 = c1; this.r2 = r2; this.c2 = c2; this.dist = dist;
    }
}

public class Solution {
    public int solution(int[][] board) {
        int n = board.length;
        // 지도의 외곽을 벽(1)으로 감싸기 위해 크기를 (n+2)로 늘립니다.
        int[][] newBoard = new int[n + 2][n + 2];
        for(int i=0; i<n+2; i++) Arrays.fill(newBoard[i], 1); // 전체를 벽으로 채웁니다.
        for(int i=0; i<n; i++)
            for(int j=0; j<n; j++) newBoard[i+1][j+1] = board[i][j]; // 실제 지도를 가운데에 넣습니다.

        Queue<Node> q = new LinkedList<>(); // BFS 탐색을 위한 큐
        q.add(new Node(1, 1, 1, 2, 0)); // 로봇의 시작 위치와 시간(0)을 넣습니다.
        Set<String> visited = new HashSet<>(); // 방문 체크를 위한 집합
        visited.add("1,1,1,2"); // 시작 위치 방문 처리

        while(!q.isEmpty()){
            Node curr = q.poll(); // 큐에서 현재 상태를 꺼냅니다.
            // 로봇의 어느 한 부분이라도 도착점(n, n)에 도달하면 종료합니다.
            if((curr.r1 == n && curr.c1 == n) || (curr.r2 == n && curr.c2 == n)) return curr.dist;

            // 이동하거나 회전해서 갈 수 있는 다음 위치들을 확인합니다.
            for(Node next : getNext(curr, newBoard)) {
                String key = makeKey(next); // 두 칸의 위치를 하나로 묶어 고유한 문자열 키를 만듭니다.
                if(!visited.contains(key)){
                    visited.add(key); // 방문 표시
                    q.add(next); // 큐에 삽입
                }
            }
        }
        return 0;
    }

    // 로봇이 현재 위치에서 다음으로 이동 가능한 모든 위치를 찾는 함수입니다.
    private List<Node> getNext(Node curr, int[][] board) {
        List<Node> next = new ArrayList<>();
        // 1. 상하좌우 이동 처리
        int[] dr = {-1, 1, 0, 0}, dc = {0, 0, -1, 1};
        for(int i=0; i<4; i++){
            int nr1 = curr.r1+dr[i], nc1 = curr.c1+dc[i], nr2 = curr.r2+dr[i], nc2 = curr.c2+dc[i];
            if(board[nr1][nc1] == 0 && board[nr2][nc2] == 0)
                next.add(new Node(nr1, nc1, nr2, nc2, curr.dist+1));
        }

        // 2. 회전 처리
        if(curr.r1 == curr.r2){ // 로봇이 가로로 놓여있는 경우
            for(int i : new int[]{-1, 1}){
                // 위쪽 또는 아래쪽 두 칸이 모두 비어있어야 회전 가능합니다.
                if(board[curr.r1+i][curr.c1] == 0 && board[curr.r2+i][curr.c2] == 0){
                    next.add(new Node(curr.r1, curr.c1, curr.r1+i, curr.c1, curr.dist+1));
                    next.add(new Node(curr.r2, curr.c2, curr.r2+i, curr.c2, curr.dist+1));
                }
            }
        } else { // 로봇이 세로로 놓여있는 경우
            for(int i : new int[]{-1, 1}){
                // 왼쪽 또는 오른쪽 두 칸이 모두 비어있어야 회전 가능합니다.
                if(board[curr.r1][curr.c1+i] == 0 && board[curr.r2][curr.c2+i] == 0){
                    next.add(new Node(curr.r1, curr.c1, curr.r1, curr.c1+i, curr.dist+1));
                    next.add(new Node(curr.r2, curr.c2, curr.r2, curr.c2+i, curr.dist+1));
                }
            }
        }
        return next;
    }

    // 두 좌표의 순서가 바뀌어도 같은 위치로 인식하도록 문자열 키를 만드는 함수입니다.
    private String makeKey(Node n) {
        int r1 = n.r1, c1 = n.c1, r2 = n.r2, c2 = n.c2;
        // 항상 좌표값이 작은 쪽을 앞으로 오게 하여 동일성을 보장합니다.
        if(r1 > r2 || (r1 == r2 && c1 > c2)) {
            int tr = r1, tc = c1; r1 = r2; c1 = c2; r2 = tr; c2 = tc;
        }
        return r1 + "," + c1 + "," + r2 + "," + c2;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in); // 입력을 받기 위한 준비
        Solution sol = new Solution();

        System.out.print("지도의 크기(n)를 입력하세요: ");
        int n = sc.nextInt();
        int[][] board = new int[n][n];

        System.out.println("지도를 입력하세요 (0: 빈칸, 1: 벽):");
        for(int i=0; i<n; i++) {
            for(int j=0; j<n; j++) {
                board[i][j] = sc.nextInt();
            }
        }
        System.out.println("결과: " + sol.solution(board));
    }
}
```

## 핵심 포인트

- **좌표 정렬**: Java나 C++에서 방문 처리를 위해 좌표 세트를 키로 사용할 때, (r1, c1)과 (r2, c2)의 순서에 상관없이 동일한 위치로 인식하도록 정렬하는 과정이 중요합니다.
- **BFS 최단 거리**: 큐를 이용하여 가장 먼저 목적지에 도달하는 경로가 최단 시간임을 보장합니다.
