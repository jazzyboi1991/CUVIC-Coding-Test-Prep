# [해설] 신규 아이디 추천

## 문제 분석

이 문제는 카카오의 신규 아이디 생성 규칙에 따라 입력받은 아이디를 정해진 7단계의 과정을 거쳐 변환하는 **구현(Implementation)** 및 **문자열 처리(String Manipulation)** 문제입니다.

**핵심 난이도**:

- 각 단계의 규칙이 이전 단계의 결과물에 순차적으로 적용되어야 한다는 점입니다.
- 특히 정규표현식을 사용하지 않을 경우 2단계(특수문자 제거)와 3단계(연속 마침표 처리)에서 코드가 복잡해질 수 있습니다.
- 6단계에서 문자열을 자른 후 다시 마지막에 마침표가 생기는 예외 상황을 잘 처리해야 합니다.

## 단계별 풀이 전략 (상세)

1. **1단계 (소문자화)**:
    - 입력된 `new_id`에서 모든 대문자를 소문자로 바꿉니다. 이는 이후 비교를 단순화하기 위함입니다.
2. **2단계 (필터링)**:
    - 소문자, 숫자, 빼기(`-`), 밑줄(`_`), 마침표(`.`)를 제외한 모든 문자를 제거합니다.
    - 정규표현식 `[^a-z0-9\-_.]`을 사용하면 한 줄로 해결 가능하지만, 반복문으로 처리할 경우 각 문자의 ASCII 값이나 `isalnum()` 등을 활용해야 합니다.
3. **3단계 (연속 마침표 축약)**:
    - 마침표(`.`)가 2번 이상 연속된 부분을 하나의 마침표로 바꿉니다.
    - 예: `...` -> `.` / `..` -> `.`
    - 정규표현식 `\.+` 또는 `[.]+`을 사용하면 편리합니다.
4. **4단계 (양 끝 마침표 제거)**:
    - 문자열의 가장 앞이나 뒤에 마침표가 있다면 제거합니다.
    - Python의 `strip('.')`, Java의 정규표현식 `^[.]|[.]$` 등을 활용합니다.
5. **5단계 (빈 문자열 처리)**:
    - 만약 4단계까지 거친 결과가 빈 문자열이라면, 기본값인 `"a"`를 대입합니다.
6. **6단계 (길이 제한)**:
    - 문자열의 길이가 16자 이상이면 앞의 15자만 남기고 나머지는 버립니다.
    - **중요**: 자른 후 마지막 문자가 마침표가 되었다면, 다시 한 번 제거해줘야 합니다 (4단계의 규칙 재적용).
7. **7단계 (최소 길이 보장)**:
    - 문자열의 길이가 2자 이하라면, 길이가 3이 될 때까지 마지막 문자를 끝에 덧붙입니다.
    - 예: `"z"` -> `"zzz"` / `"co"` -> `"coo"`

## 코드 구현

### 1. Python

```python
import re

def solution(new_id):
    # 1단계: 소문자 치환
    st = new_id.lower()
    # 2단계: 허용되지 않는 문자 제거
    st = re.sub('[^a-z0-9\-_.]', '', st)
    # 3단계: 연속된 마침표 하나로 치환
    st = re.sub('\.+', '.', st)
    # 4단계: 처음이나 끝에 위치한 마침표 제거
    st = st.strip('.')
    # 5단계: 빈 문자열이라면 "a" 대입
    if not st:
        st = "a"
    # 6단계: 길이가 16자 이상이면 15자까지 자르고 끝 마침표 제거
    if len(st) >= 16:
        st = st[:15]
        st = st.rstrip('.')
    # 7단계: 길이가 2자 이하라면 마지막 문자 반복
    while len(st) <= 2:
        st += st[-1]
    return st

# 터미널 입력 예시
if __name__ == "__main__":
    new_id = input("새로운 아이디를 입력하세요: ")
    print(f"추천 아이디: {solution(new_id)}")
```

### 2. C++

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <algorithm>

using namespace std;

string solution(string new_id) {
    // 1단계
    for (char &c : new_id) c = tolower(c);

    // 2단계
    string st = "";
    for (char c : new_id) {
        if (islower(c) || isdigit(c) || c == '-' || c == '_' || c == '.') st += c;
    }

    // 3단계
    string temp = "";
    for (char c : st) {
        if (c == '.') {
            if (temp.length() > 0 && temp.back() == '.') continue;
        }
        temp += c;
    }
    st = temp;

    // 4단계
    if (st.front() == '.') st.erase(st.begin());
    if (st.back() == '.') st.pop_back();

    // 5단계
    if (st.empty()) st = "a";

    // 6단계
    if (st.length() >= 16) {
        st = st.substr(0, 15);
        if (st.back() == '.') st.pop_back();
    }

    // 7단계
    while (st.length() <= 2) st += st.back();

    return st;
}

int main() {
    string new_id;
    cout << "새로운 아이디를 입력하세요: ";
    cin >> new_id;
    cout << "추천 아이디: " << solution(new_id) << endl;
    return 0;
}
```

### 3. Java

```java
import java.util.Scanner; // 사용자로부터 입력을 받기 위한 스캐너 도구를 가져옵니다.

public class Solution {
    public String solution(String new_id) {
        // 1단계 & 2단계 & 3단계: 소문자 변환, 허용된 문자 외 제거, 연속 마침표 축약
        String answer = new_id.toLowerCase()
                .replaceAll("[^-_.a-z0-9]", "")
                .replaceAll("[.]{2,}", ".");

        // 4단계: 처음이나 끝에 위치한 마침표 제거
        answer = answer.replaceAll("^[.]|[.]$", "");

        // 5단계: 빈 문자열이라면 "a" 대입
        if (answer.isEmpty()) answer = "a";

        // 6단계: 길이가 16자 이상이면 15자까지 자르고 끝 마침표 제거
        if (answer.length() >= 16) {
            answer = answer.substring(0, 15).replaceAll("[.]$", "");
        }

        // 7단계: 길이가 2자 이하라면 마지막 문자 반복하여 길이 3 충족
        while (answer.length() <= 2) {
            answer += answer.charAt(answer.length() - 1);
        }

        return answer;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in); // 입력 도구 생성
        Solution sol = new Solution();

        System.out.print("새로운 아이디를 입력하세요: ");
        String new_id = sc.next();
        System.out.println("추천 아이디: " + sol.solution(new_id));
    }
}
```

## 핵심 포인트

- **정규표현식 활용**: Python과 Java에서는 정규표현식을 통해 2~4단계를 매우 효율적으로 처리할 수 있습니다.
- **문자열 조작**: C++에서는 `substr`, `erase`, `back` 등의 함수를 사용하여 직접 문자를 제어하는 방식이 직관적입니다.
- **순차적 처리**: 각 단계의 규칙이 이전 단계의 결과에 영향을 받으므로 순서를 지키는 것이 매우 중요합니다.
