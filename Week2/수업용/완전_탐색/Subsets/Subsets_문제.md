# [LeetCode] 78. Subsets

## 1. 문제 설명
고유한 정수들로 이루어진 배열 `nums`가 주어질 때, 가능한 모든 부분 집합(멱집합, power set)을 반환해야 합니다.

해답 집합에는 중복된 부분 집합이 포함되어서는 안 되며, 결과는 어떤 순서로든 반환할 수 있습니다.

## 2. 제한 사항
- `1 <= nums.length <= 10`
- `-10 <= nums[i] <= 10`
- `nums`의 모든 숫자는 **고유(unique)**합니다.

## 3. 입출력 예

**예제 1**
- **입력**: `1 2 3`
- **출력**: `[[], [1], [1, 2], [1, 2, 3], [1, 3], [2], [2, 3], [3]]`

**예제 2**
- **입력**: `0`
- **출력**: `[[], [0]]`

## 4. 입출력 예 설명

### 예제 1
- **입력**: `1 2 3`
- **출력**: `[[], [1], [1, 2], [1, 2, 3], [1, 3], [2], [2, 3], [3]]`
- **설명**: 원소가 3개인 집합의 부분 집합은 총 $2^3 = 8$개입니다. 빈 집합 `[]`을 포함하여 가능한 모든 조합을 나열합니다.

### 예제 2
- **입력**: `0`
- **출력**: `[[], [0]]`
- **설명**: 원소가 1개인 집합의 부분 집합은 빈 집합과 자기 자신을 포함하여 총 2개입니다.
