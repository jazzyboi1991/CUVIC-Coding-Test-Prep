# [LeetCode] 77. Combinations

## 1. 문제 설명
두 개의 정수 `n`과 `k`가 주어졌을 때, 1부터 `n` 사이의 범위에서 `k`개의 숫자를 선택하여 만들 수 있는 모든 가능한 조합을 반환해야 합니다.

결과값의 순서는 관계없습니다.

## 2. 제한 사항
- `1 <= n <= 20`
- `1 <= k <= n`

## 3. 입출력 예

**예제 1**
- **입력**: `4 2`
- **출력**: `[[1, 2], [1, 3], [1, 4], [2, 3], [2, 4], [3, 4]]`

**예제 2**
- **입력**: `1 1`
- **출력**: `[[1]]`

## 4. 입출력 예 설명

### 예제 1
- **입력**: `4 2`
- **출력**: `[[1, 2], [1, 3], [1, 4], [2, 3], [2, 4], [3, 4]]`
- **설명**: 4개(1, 2, 3, 4) 중 2개를 선택하는 총 6가지 조합이 있습니다. 조합은 순서가 없으므로 `[1, 2]`와 `[2, 1]`은 동일한 조합으로 간주됩니다.

### 예제 2
- **입력**: `1 1`
- **출력**: `[[1]]`
- **설명**: 1개(1) 중 1개를 선택하는 총 1가지 조합이 있습니다.
