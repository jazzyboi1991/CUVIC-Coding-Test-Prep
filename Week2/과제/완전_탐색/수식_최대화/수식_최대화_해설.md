# [2020 카카오 인턴십] 수식 최대화 - 해설

## 1. 문제 분석
이 문제는 주어진 연산자들의 우선순위를 임의로 조정하여 계산된 결과의 절댓값을 최대화하는 문제입니다.

- **알고리즘 유형**: 완전 탐색 (Brute Force), 구현 (Implementation)
- **핵심 포인트**: 
  - 연산자가 `+`, `-`, `*` 세 종류뿐이므로, 가능한 우선순위 조합은 총 $3! = 6$가지밖에 되지 않습니다.
  - 모든 조합에 대해 계산을 수행해보고 가장 큰 값을 찾으면 됩니다.
  - 연산 도중 결과가 매우 커질 수 있으므로, 64비트 정수형(long long)을 사용해야 합니다.

## 2. 단계별 풀이 전략

1. **연산자 우선순위 생성**:
   - `+`, `-`, `*` 세 연산자의 모든 순열(Permutation)을 생성합니다. (총 6가지)

2. **수식 파싱 (Parsing)**:
   - 숫자와 연산자를 각각의 리스트(혹은 벡터)에 분리하여 담습니다.
   - 예: `"100-200*300"` → 숫자 리스트 `[100, 200, 300]`, 연산자 리스트 `[-, *]`

3. **우선순위에 따른 계산 시뮬레이션**:
   - 각 우선순위 조합에 대해 다음을 반복합니다:
     - 현재 우선순위가 가장 높은 연산자부터 수식에서 찾습니다.
     - 해당 연산자가 연산자 리스트에 있다면, 그 위치의 숫자 두 개를 계산합니다.
     - 계산된 결과로 숫자 리스트를 업데이트하고, 사용한 연산자는 리스트에서 제거합니다.

4. **결과 비교**:
   - 모든 계산이 끝난 후 남은 하나의 숫자에 절댓값을 씌워 현재까지의 최댓값과 비교합니다.

## 3. 코드 구현

### Python
```python
import sys  # 표준 입력을 위해 사용합니다.
from itertools import permutations # 모든 순열 조합을 만들기 위해 사용합니다.

def calculate(n1, n2, op):
    """연산자에 맞춰 두 숫자를 계산하는 함수입니다."""
    if op == '+': return n1 + n2
    if op == '-': return n1 - n2
    if op == '*': return n1 * n2

def solution(expression):
    # 1. 숫자와 연산자를 분리합니다.
    nums = []
    ops = []
    temp = ""
    for char in expression:
        if char.isdigit():
            temp += char
        else:
            nums.append(int(temp))
            ops.append(char)
            temp = ""
    nums.append(int(temp)) # 마지막 숫자 추가
    
    # 2. 가능한 모든 연산자 우선순위 조합을 만듭니다.
    max_reward = 0
    for priority in permutations(['+', '-', '*']):
        # 원본 데이터를 복사해서 사용합니다.
        curr_nums = nums[:]
        curr_ops = ops[:]
        
        # 3. 우선순위대로 연산을 수행합니다.
        for p_op in priority:
            i = 0
            while i < len(curr_ops):
                if curr_ops[i] == p_op:
                    # 해당 연산자와 양옆의 숫자를 계산합니다.
                    res = calculate(curr_nums[i], curr_nums[i+1], p_op)
                    # 계산 결과로 교체하고 사용한 데이터는 삭제합니다.
                    curr_nums[i] = res
                    curr_nums.pop(i + 1)
                    curr_ops.pop(i)
                else:
                    i += 1
        
        # 4. 결과값의 절댓값 중 최댓값을 찾습니다.
        max_reward = max(max_reward, abs(curr_nums[0]))
        
    return max_reward

if __name__ == "__main__":
    # 사용자로부터 수식을 입력받습니다.
    expr = sys.stdin.readline().strip()
    # 결과 출력
    print(f"최대 상금: {solution(expr)}")
```

### C++
```cpp
#include <iostream>  // 표준 입출력을 위한 라이브러리입니다.
#include <vector>    // 가변 배열 사용을 위해 필요합니다.
#include <string>    // 문자열 처리를 위해 필요합니다.
#include <algorithm> // next_permutation, abs 함수 등을 위해 사용합니다.

#define endl "\n"

using namespace std;

/**
 * [calc 함수 설명]
 * 두 숫자와 연산자를 받아 계산 결과를 반환합니다.
 * 큰 숫자 처리를 위해 long long 타입을 사용합니다.
 */
long long calc(long long a, long long b, char op) {
    if (op == '+') return a + b;
    if (op == '-') return a - b;
    if (op == '*') return a * b;
    return 0;
}

/**
 * [solution 함수 설명]
 * 모든 연산자 우선순위 조합을 시도하여 최대 절댓값을 찾습니다.
 */
long long solution(string expression) {
    vector<long long> nums; // 숫자들을 담을 벡터
    vector<char> ops;       // 연산자들을 담을 벡터
    
    // 1. 수식 파싱
    string temp = "";
    for (char c : expression) {
        if (isdigit(c)) {
            temp += c;
        } else {
            nums.push_back(stoll(temp)); // 문자열을 long long으로 변환
            ops.push_back(c);
            temp = "";
        }
    }
    nums.push_back(stoll(temp));

    // 2. 연산자 우선순위 순열 준비
    vector<char> priority = {'*', '+', '-'};
    sort(priority.begin(), priority.end());
    
    long long max_val = 0;

    // 3. 모든 순열 조합에 대해 계산 수행
    do {
        vector<long long> temp_nums = nums;
        vector<char> temp_ops = ops;

        for (char p_op : priority) {
            for (int i = 0; i < temp_ops.size(); ) {
                if (temp_ops[i] == p_op) {
                    long long res = calc(temp_nums[i], temp_nums[i + 1], p_op);
                    temp_nums[i] = res;
                    temp_nums.erase(temp_nums.begin() + i + 1);
                    temp_ops.erase(temp_ops.begin() + i);
                } else {
                    i++;
                }
            }
        }
        // 최댓값 갱신
        max_val = max(max_val, abs(temp_nums[0]));
        
    } while (next_permutation(priority.begin(), priority.end()));

    return max_val;
}

int main() {
    string s;
    cout << "수식을 입력하세요: ";
    cin >> s;
    cout << "최대 상금: " << solution(s) << endl;
    return 0;
}
```

### Java
```java
import java.util.*; // ArrayList, List 등을 사용하기 위해 불러옵니다.

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in); // 입력 도구 생성
        System.out.print("수식을 입력하세요: ");
        String expression = sc.next();
        
        System.out.println("최대 상금: " + solution(expression));
        sc.close();
    }

    /**
     * 실제 계산을 수행하는 함수입니다.
     */
    public static long calculate(long a, long b, String op) {
        if (op.equals("+")) return a + b;
        if (op.equals("-")) return a - b;
        return a * b;
    }

    public static long solution(String expression) {
        // 1. 숫자와 연산자 분리
        List<Long> nums = new ArrayList<>();
        List<String> ops = new ArrayList<>();
        
        StringBuilder sb = new StringBuilder();
        for (char c : expression.toCharArray()) {
            if (Character.isDigit(c)) {
                sb.append(c);
            } else {
                nums.add(Long.parseLong(sb.toString()));
                ops.add(String.valueOf(c));
                sb.setLength(0); // 빌더 초기화
            }
        }
        nums.add(Long.parseLong(sb.toString()));

        // 2. 가능한 6가지 우선순위 조합 정의
        String[][] priorities = {
            {"*", "+", "-"}, {"*", "-", "+"},
            {"+", "*", "-"}, {"+", "-", "*"},
            {"-", "*", "+"}, {"-", "+", "*"}
        };

        long maxVal = 0;

        // 3. 각 조합마다 시뮬레이션 수행
        for (String[] priority : priorities) {
            List<Long> tempNums = new ArrayList<>(nums);
            List<String> tempOps = new ArrayList<>(ops);

            for (String pOp : priority) {
                for (int i = 0; i < tempOps.size(); ) {
                    if (tempOps[i].equals(pOp)) {
                        long res = calculate(tempNums.get(i), tempNums.get(i + 1), pOp);
                        tempNums.set(i, res); // 결과로 교체
                        tempNums.remove(i + 1); // 뒤의 숫자 제거
                        tempOps.remove(i); // 연산자 제거
                    } else {
                        i++;
                    }
                }
            }
            // 절댓값 중 최대값 갱신
            maxVal = Math.max(maxVal, Math.abs(tempNums.get(0)));
        }

        return maxVal;
    }
}
```

## 4. 핵심 포인트 및 주의 사항
- **자료형 선택**: 중간 계산 과정에서 값이 매우 커질 수 있으므로 C++의 `long long`, Java의 `long`, Python의 큰 정수 처리를 활용해야 합니다.
- **리스트 삭제 시 인덱스 주의**: 계산이 끝난 후 `remove`나 `erase`를 수행하면 리스트의 크기가 줄어들므로, 인덱스 관리를 신중히 하거나 루프 조건을 잘 설정해야 합니다. (계산이 일어나지 않았을 때만 `i++`를 수행하는 방식이 안전합니다.)
- **절댓값 처리**: 최종 결과에 `abs()` 함수를 적용하여 상금을 계산하는 것을 잊지 마세요.
- **순열(Permutation)**: 연산자가 3개이므로 직접 6가지 경우를 코딩해도 되지만, 언어에서 제공하는 순열 기능을 활용하면 실수를 줄일 수 있습니다.
