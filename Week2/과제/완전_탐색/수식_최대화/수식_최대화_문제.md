# [2020 카카오 인턴십] 수식 최대화

## 1. 문제 설명
IT 벤처 회사를 운영하고 있는 라이언은 매년 사내 해커톤 대회를 개최하여 우승자에게 상금을 지급하고 있습니다. 이번 대회에서는 전달받은 수식에 포함된 연산자의 우선순위를 자유롭게 재정의하여 만들 수 있는 가장 큰 숫자를 제출하는 미션을 주었습니다.

단, 연산자의 우선순위를 새로 정의할 때, 같은 순위의 연산자는 없어야 합니다. 수식에 포함된 연산자가 3개라면 3! = 6가지 조합이 가능합니다. 만약 계산된 결과가 음수라면 해당 숫자의 절댓값으로 변환하여 제출하며 제출한 숫자가 가장 큰 참가자를 우승자로 선정합니다.

예를 들어, "100-200*300-500+20" 수식이 주어졌을 때, 연산자 우선순위를 `* > + > -` 로 정한다면 최종 상금은 60,420원이 됩니다.

참가자에게 주어진 연산 수식이 담긴 문자열 `expression`이 매개변수로 주어질 때, 우승 시 받을 수 있는 가장 큰 상금 금액을 return 하도록 solution 함수를 완성해주세요.

## 2. 제한사항
- `expression`은 길이가 3 이상 100 이하인 문자열입니다.
- `expression`은 숫자와 3가지의 연산자(+, -, *) 만으로 이루어진 올바른 중위표기법 수식입니다.
- 피연산자(숫자)는 0 이상 999 이하의 정수입니다.
- 연산 결과 및 중간 계산값은 절댓값이 $2^{63} - 1$ 이하(64비트 정수 범위)가 되도록 입력이 주어집니다.
- 같은 연산자끼리는 앞에 있는 것의 우선순위가 더 높습니다.

##### **[입출력 예]**

**예제 1**
- **입력**
  - 수식(expression): `100-200*300-500+20`
- **출력**: `60420`

**예제 2**
- **입력**
  - 수식(expression): `50*6-3*2`
- **출력**: `300`

## 4. 입출력 예 설명

### 예제 1
`* > + > -` 로 연산자 우선순위를 정했을 때 가장 큰 절댓값을 얻을 수 있습니다.
1. `200 * 300 = 60,000` 계산 → "100-60000-500+20"
2. `500 + 20 = 520` 계산 → "100-60000-520"
3. `100 - 60000 = -59,900` 계산 → "-59900-520"
4. `-59900 - 520 = -60,420` 계산
최종적으로 $|-60,420| = 60,420$원이 상금이 됩니다.

### 예제 2
`- > *` 로 연산자 우선순위를 정했을 때 가장 큰 절댓값 300을 얻을 수 있습니다. (+ 연산자가 없는 경우 나머지 연산자들로만 순위를 정합니다.)
1. `6 - 3 = 3` 계산 → "50 * 3 * 2"
2. `50 * 3 = 150` 계산 → "150 * 2"
3. `150 * 2 = 300` 계산
따라서 300이 상금이 됩니다.
