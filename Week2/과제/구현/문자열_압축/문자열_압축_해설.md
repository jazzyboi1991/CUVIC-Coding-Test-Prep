# [2020 KAKAO BLIND RECRUITMENT] 문자열 압축 - 해설

## 1. 문제 분석
이 문제는 문자열을 일정한 단위(1, 2, 3...)로 잘랐을 때, 연속되는 반복 문자열을 숫자로 치환하여 압축된 문자열의 최소 길이를 찾는 문제입니다.

- **알고리즘 유형**: 구현 (Implementation), 완전 탐색 (Brute Force)
- **시간 복잡도**: $O(N^2)$ - 문자열의 길이 $N$에 대해 단위를 $1$부터 $N/2$까지 시도하고, 각 단위마다 문자열 전체를 훑으므로 약 $N \times (N/step)$번 연산합니다. $N$이 최대 1,000이므로 충분히 해결 가능합니다.
- **핵심 포인트**: 
  - 문자열을 앞에서부터 일정한 길이만큼씩 잘라야 합니다. (중간부터 자르기 시작하는 것은 허용되지 않음)
  - 압축 단위는 1부터 문자열 전체 길이의 절반($N/2$)까지만 고려하면 됩니다. (절반보다 크면 반복될 수 없기 때문)

## 2. 단계별 풀이 전략

1. **압축 단위 결정**:
   - `step`이라는 변수를 1부터 `len(s)//2`까지 1씩 증가시키며 반복문을 돌립니다.
   - 단, 문자열 길이가 1인 경우는 압축 자체가 불가능하므로 예외 처리가 필요합니다.

2. **문자열 자르기 및 비교**:
   - 현재 단위(`step`)만큼 첫 조각을 떼어내어 `prev` 변수에 저장합니다.
   - 그 다음 조각부터 차례대로 비교하며, `prev`와 같다면 개수(`count`)를 1 늘립니다.
   - 만약 다른 조각이 나타나면, 지금까지 센 `count`와 `prev`를 압축 문자열에 기록하고 `prev`를 새 조각으로 바꿉니다. (이때 `count`가 1이면 숫자는 생략합니다.)

3. **최종 결과 도출**:
   - 반복문이 끝난 후 마지막에 남아있던 조각까지 기록한 뒤, 압축된 문자열의 길이를 구합니다.
   - 모든 `step`에 대해 계산한 길이 중 가장 작은 값을 정답으로 채택합니다.

## 3. 코드 구현

### Python
```python
import sys  # 표준 입출력을 위해 사용합니다.

def solution(s):
    # 문자열 길이가 1이면 더 이상 줄어들 수 없으므로 길이는 1입니다.
    if len(s) == 1:
        return 1
    
    # 정답을 저장할 변수입니다. 초기값은 압축 전의 최대 길이로 설정합니다.
    answer = len(s)
    
    # 1개 단위부터 문자열의 절반 길이 단위까지 시도해봅니다.
    for step in range(1, len(s) // 2 + 1):
        compressed = ""  # 압축된 결과 문장을 담을 변수입니다.
        prev = s[0:step] # 맨 앞의 첫 번째 조각을 미리 가져옵니다.
        count = 1        # 같은 조각이 나타난 횟수입니다.
        
        # step 크기만큼 건너뛰면서 다음 조각들을 확인합니다.
        for j in range(step, len(s), step):
            # 이전 조각(prev)과 현재 조각이 같다면
            if prev == s[j:j+step]:
                count += 1 # 횟수를 하나 늘립니다.
            # 다른 조각이 나타났다면
            else:
                # 횟수가 2번 이상이라면 숫자와 글자를 함께 적습니다.
                # 1번뿐이라면 숫자 없이 글자만 적습니다.
                compressed += str(count) + prev if count >= 2 else prev
                # 비교할 기준(prev)을 현재 조각으로 바꾸고 횟수를 1로 초기화합니다.
                prev = s[j:j+step]
                count = 1
        
        # 마지막에 남아있는 조각까지 압축 문장에 추가합니다.
        compressed += str(count) + prev if count >= 2 else prev
        
        # 만들어진 압축 문장의 길이와 현재까지의 최솟값을 비교합니다.
        answer = min(answer, len(compressed))
        
    return answer

if __name__ == "__main__":
    # 사용자로부터 문자열을 입력받습니다.
    input_str = sys.stdin.readline().strip()
    # 결과를 출력합니다.
    print(f"최소 압축 길이: {solution(input_str)}")
```

### C++
```cpp
#include <iostream> // 화면 출력과 입력을 위한 라이브러리입니다.
#include <string>   // 문자열을 편하게 다루기 위해 사용합니다.
#include <vector>   // 가변 배열을 다루기 위해 사용합니다.
#include <algorithm> // min 함수 등을 사용하기 위해 불러옵니다.

#define endl "\n"    // 줄 바꿈을 빠르게 처리하기 위한 설정입니다.

using namespace std; // 표준 라이브러리의 이름을 편하게 사용합니다.

/**
 * [solution 함수 설명]
 * 문자열을 1부터 길이의 절반까지 단위별로 압축하여 가장 짧은 길이를 찾습니다.
 * @param s: 압축할 원본 문자열입니다.
 * @return: 가장 짧게 압축된 문자열의 길이를 반환합니다.
 */
int solution(string s) {
    // 문자열의 길이가 1이면 압축해도 1이므로 바로 반환합니다.
    if (s.length() == 1) return 1;
    
    int answer = s.length(); // 정답의 초기값은 원래 문자열의 길이입니다.
    
    // 1부터 길이의 절반까지 단위를 늘려가며 확인합니다.
    for (int step = 1; step <= s.length() / 2; step++) {
        string compressed = ""; // 압축된 결과를 저장할 변수입니다.
        string prev = s.substr(0, step); // 첫 번째 조각을 잘라옵니다.
        int count = 1; // 반복 횟수를 저장합니다.
        
        // step 크기만큼 건너뛰며 다음 글자들과 비교합니다.
        for (int j = step; j < s.length(); j += step) {
            // 이전 조각과 현재 조각이 같다면 횟수를 증가시킵니다.
            if (prev == s.substr(j, step)) {
                count++;
            } 
            // 서로 다르다면 지금까지의 결과를 압축 문자열에 붙입니다.
            else {
                if (count >= 2) compressed += to_string(count); // 2번 이상이면 숫자 추가
                compressed += prev; // 글자 추가
                
                prev = s.substr(j, step); // 새로운 조각을 기준으로 설정
                count = 1; // 횟수 초기화
            }
        }
        
        // 반복문이 끝난 후 처리되지 않은 마지막 조각을 추가합니다.
        if (count >= 2) compressed += to_string(count);
        compressed += prev;
        
        // 지금까지 구한 최솟값과 현재 압축 길이를 비교합니다.
        answer = min(answer, (int)compressed.length());
    }
    
    return answer;
}

int main() {
    string input_s;
    cout << "압축할 문자열을 입력하세요: ";
    cin >> input_s; // 사용자로부터 문자열을 입력받습니다.
    
    // 결과 출력
    cout << "최소 압축 길이: " << solution(input_s) << endl;
    return 0;
}
```

### Java
```java
import java.util.Scanner; // 키보드 입력을 받기 위해 사용합니다.

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in); // 입력 도구 생성
        
        System.out.print("압축할 문자열을 입력하세요: ");
        if (sc.hasNextLine()) {
            String s = sc.nextLine(); // 문자열 한 줄을 입력받습니다.
            System.out.println("최소 압축 길이: " + solution(s)); // 결과 출력
        }
        sc.close(); // 입력 도구 닫기
    }

    /**
     * [solution 함수 설명]
     * 입력받은 문자열을 다양한 단위로 잘라보고 가장 짧은 압축 길이를 계산합니다.
     * @param s: 입력받은 문자열
     * @return: 가장 짧게 압축된 결과의 길이
     */
    public static int solution(String s) {
        // 문자열 길이가 1인 경우 예외 처리
        if (s.length() == 1) return 1;
        
        int answer = s.length(); // 초기값은 최대 길이인 원본 길이로 설정
        
        // 압축 단위(step)를 1부터 절반까지 시도
        for (int step = 1; step <= s.length() / 2; step++) {
            StringBuilder compressed = new StringBuilder(); // 효율적인 문자열 합치기를 위해 StringBuilder 사용
            String prev = s.substring(0, step); // 첫 조각 추출
            int count = 1; // 반복 횟수
            
            // step 만큼 인덱스를 옮겨가며 조각 비교
            for (int j = step; j < s.length(); j += step) {
                // 다음 조각의 끝 위치 계산 (문자열 범위를 넘지 않도록 주의)
                int end = Math.min(j + step, s.length());
                String current = s.substring(j, end);
                
                // 이전 조각과 같으면 횟수 증가
                if (prev.equals(current)) {
                    count++;
                } 
                // 다르면 지금까지의 내용 기록
                else {
                    if (count >= 2) compressed.append(count); // 2번 이상 반복 시 숫자 추가
                    compressed.append(prev); // 조각 글자 추가
                    
                    prev = current; // 기준 조각 변경
                    count = 1; // 횟수 초기화
                }
            }
            
            // 마지막 남은 조각 처리
            if (count >= 2) compressed.append(count);
            compressed.append(prev);
            
            // 현재 step으로 압축한 결과의 길이와 최솟값 비교
            answer = Math.min(answer, compressed.length());
        }
        
        return answer;
    }
}
```

## 4. 핵심 포인트 및 주의 사항
- **단위의 범위**: 문자열 길이 $N$이 홀수인 경우나 짝수인 경우 모두 $N/2$까지만 확인하면 됩니다. 예를 들어 $N=5$일 때 3개 단위로 자르면 반복될 수 없습니다.
- **문자열 자르기 주의**: Java의 `substring`이나 C++의 `substr`을 사용할 때, 마지막 조각이 단위보다 짧게 남는 경우 인덱스 범위를 초과하지 않도록 조심해야 합니다. (Java의 경우 `Math.min(j + step, s.length())` 등으로 처리)
- **문자열 길이 1**: 입력 문자열이 "a"와 같이 1글자라면 반복문이 돌지 않으므로, 반드시 초기에 예외 처리를 해주어야 합니다.
